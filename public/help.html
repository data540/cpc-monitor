CPC Monitor

Dashboard de monitorización en tiempo real para campañas de Google Ads con estrategia Target ROAS. Controla el CPC ceiling, el Impression Share, el ROAS real y la presencia competitiva en la subasta desde un único panel.

Config
/dashboard/config

Panel de configuración del sistema para personalizar el comportamiento del monitor.

  • 🆔Customer ID por defecto: Define qué cuenta de Google Ads se carga automáticamente al abrir el dashboard
  • 💰Techos CPC manuales: Cuando Google Ads no expone el ceiling a través de la API, puedes introducirlo manualmente por nombre exacto de campaña
  • 📏Umbrales IS: Configura el umbral de IS para las alertas de campaña de marca (por defecto 85% para campañas con "marca" o "brand" en el nombre)

Dashboard
  • 🔑Autenticación: Google OAuth2 con scope https://www.googleapis.com/auth/adwords. El token se refresca automáticamente cuando caduca.
  • 🔄Refresco de token: Si el refresh_token es revocado (invalid_grant), el sistema limpia los tokens y redirige a re-autenticación.
  • 🗄Históricos: Cada carga de campañas guarda un snapshot en PostgreSQL para los gráficos de evolución de 30 días.
  • 🏗MCC + login-customer-id: El header login-customer-id identifica el MCC manager para poder acceder a cuentas hijo y estrategias de cartera.
